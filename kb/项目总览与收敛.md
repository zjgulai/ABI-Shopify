## 5. 已知边界(诚实记录)
- **外部站点边界**:YouTube 已可通过用户授权的真实 Chrome browser-harness 做只读页面复核、UI 转写入口检查与采样帧截图;GitHub/Accio/Reddit 等仍受域名策略或站点限制,**不绕过**。→ 解决:browser-harness 可用时做内容级 v0.1;否则走用户**粘贴清单/字幕/帖子**。
- **视频深度萃取**:逐字稿级仍需字幕或稳定转写;仅靠标题只能编目。当前已有 3 条内容级萃取,其中 Ac Hampton `vXmF10ZNmoo` 是 browser-harness 页面/章节/采样帧 v0.1,非完整逐字稿。
- **T6 多源入库**:执行队列和离线草稿工具已落地;没有用户提供的新字幕/帖子清单时,不会新增真实 Source/条目到图谱。
- **inbox 资料包**:已按本地只读方式入库;`listing优化模板.xlsx` 的关键词库仅做结构与样本级抽取,若要全量关键词资产化需另跑结构化导出。
- **Reddit r/shopify**:直抓被屏蔽;已用可核实来源做"同类痛点"综合(非逐字)。
- **T7 受控写**:Runbook 已落地;真实读店铺/写测试商品仍需用户完成 `shopify auth login` 测试店授权并逐次批准 mutation。
- **需用户亲自执行或确认**:页面手动录入 DeepSeek API Key 做真实 provider 问答、Accio 登录/下单/付款、测试店授权与任何资金/凭据动作;OpenAI embedding 未启用。
- **检索模型边界**:腾讯云轻量服务器已启用 `BAAI/bge-small-zh-v1.5 + Chroma + Neo4j`;`bge-m3` 在当前 CPU 资源下未上线,不能对外宣称生产使用 bge-m3。
