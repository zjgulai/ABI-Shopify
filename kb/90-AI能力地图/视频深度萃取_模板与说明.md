# 🎬 视频深度内容萃取 · 模板与说明

## 现状(诚实说明)
- 已完成:**标题/链接/发布时间的编目 + 按标题归类到节点**(见 Kevin Stratvert / Metics Media / Ac Hampton 三份目录)。
- **未完成**:看视频内容、逐字稿、内容级深度总结。
- 原因:YouTube 内容无法在本工具链合规获取(扩展域名拦截 + 反爬;不使用第三方下载/转写脚本)。

## 如何拿到字幕(你侧,1 分钟)
YouTube 视频 → 简介下「...更多」→「**显示转写 / Show transcript**」→ 右侧转写面板 → 全选复制 → 贴给我(可一次贴多个,注明视频标题/链接)。

## 每个视频的深度萃取结构(我产出)
1. **一句话主旨**
2. **核心步骤 / SOP**(可执行,编号)
3. **用到的工具 / App / 平台**(逐一列)
4. **关键数据 / 案例 / 数字**(标注是否营销话术)
5. **坑与反例**(作者强调的避免项)
6. **→ 对应节点**(映射到本库 00–10)
7. **可落地要点**(给我们独立站的 1–3 条 takeaway)
8. **辨别夸大**(收入承诺/夸张结论的客观提示)

产出后:写入对应节点文档的「§3/§4 萃取」区,并入图谱(Video—COVERS→主题)、重建 RAG/站点。

## 优先级建议(76 个视频不必全做)
大量入门建站/无货源视频高度重叠,建议**先深挖高价值、低重叠**的少数:
- **Claude Code + Shopify = $130,000/Month**(Ac Hampton)— AI×Shopify 商业打法
- **My AI Dropshipping Store … $200,000/mo** / **I Tried AI Dropshipping**(AI 无货源实测)
- **How To Fulfill Your Dropshipping Orders**(履约实操)
- **Shopify President: How to Win…**(战略访谈)
- **7-Hour Free Course / COMPLETE Shopify Tutorial**(全流程大课)
- **AI Built My Shopify Store in Minutes**(AI 建站实测)

> 贴上述任意视频的转写,我先做 1–2 个**样例**给你看格式与深度,再按你节奏批量推进。
