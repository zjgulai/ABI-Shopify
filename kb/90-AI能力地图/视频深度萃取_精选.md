## ① Claude Code + Shopify = $130,000/Month(Ac Hampton)→ 用 AI Agent 跑店铺
**主旨**:用 Claude Code 经 **Shopify AI Toolkit(MCP)** 直接读写店铺,把建店/上架/改主题/批量操作变成自然语言指令。

**核心步骤 / SOP(可执行)**
1. 安装 Dev MCP:`claude mcp add --transport stdio shopify-dev-mcp -- npx -y @shopify/dev-mcp@latest`
2. `shopify auth login` 授权(需 Partner 账号 + 目标店权限)。
3. 自然语言下指令(例:「为所有 tag=summer-2026 的商品建集合并设动态定价」)→ Agent 翻译为 API 调用 → 校验 → 执行。
4. AI Toolkit 提供 7 个工具:查文档 / 校验 GraphQL / 执行 store ops 等。

**工具**:Claude Code · Shopify AI Toolkit(MIT,2026-04-09)· Dev MCP · Shopify CLI;同样支持 Cursor / VS Code / Codex / Gemini CLI。

**可核实数据**:Shopify Q1 2026 财报(5/5)——AI 搜索带来的订单同比约 **13×**,为公司史上增长最快的入站渠道;AI 流量每访客收入比传统搜索高约 **37%**;TinyTwo 用 Claude Code 全程构建 ReviewMate 应用并投产(自动加入 HMAC/XSS 等安全实践)。

**坑 / 辨别夸大**:「$130,000/Month」为个人案例话术,**勿当可复制结果**;`store execute` 是真改店铺,务必**测试店先行 + 人审**。

## ② Shopify President: How to Win in the Golden Age of Entrepreneurship → 创业心法
**主旨(Harley Finkelstein)**:现在是历史上创业最好的时代——进入门槛崩塌、失败成本更低、AI 给了前所未有的触达与资源。

**关键观点**
- **失败与坚持(grit)**:成功者多有失败史,靠坚持"留在牌桌"直到跑通。
- **市场观**:相信 TAM 在扩大,"把饼做大"而非抢存量份额。
- **先受众后产品**:先建受众、沿途留"面包屑",把人带进旅程,做**粉丝**而非顾客。
- **故事/叙事/使命 = 品牌护城河**:"你无法和一个对所建之物有深刻信念的人竞争"。
- **AI 的角色**:AI 很强,但**品味、判断、鉴别力、野心**是它替代不了的。
