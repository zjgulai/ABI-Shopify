## 3. Learn With Shopify · The COMPLETE Shopify Tutorial 2026

**定位**:这条视频更像 Shopify 新手上线路径的压缩版,重点是 Sidekick 辅助建站、交易设置、营销中心、自动化和基础数据。

**核心路径**
1. 账号与商品:从基础 store setup、products、collections 到 logo、favicon、domain。
2. Sidekick/AI:用 Sidekick 辅助编辑主题、生成 launch email、处理页面与内容草稿。
3. 交易设置:payments、two-step authentication、shipping labels、test mode 和 order flow。
4. 营销与自动化:Marketing Hub、campaigns、attribution、paid ads、organic social、abandoned cart、VIP campaign。
5. 数据与 SEO:Google Analytics、Search Console、Shopify analytics、页面 performance、payment options、shipping cost visibility、trust signals。

**可沉淀 SOP**
- 在 ABI 站点新增“Sidekick 操作卡”时,应把它定位成辅助草拟和后台导航能力,不替代人审、店铺权限和测试店验证。
- 营销中心内容可转成 05/07 的最小闭环:campaign brief → attribution 标记 → automation → abandoned cart → VIP 复购。
- Test mode 与 two-step authentication 可纳入 T7 受控写之前的前置检查。

## 4. Emma Grede · Shopify President 访谈

**定位**:这是战略访谈材料,价值在创业观、品牌观和 AI 时代的人机分工,不能当成具体后台配置 SOP。

**可沉淀观点**
1. 低成本试错:创业进入门槛下降,失败成本降低,但执行纪律和复盘仍是关键。
2. 市场观:更重要的是“把市场做大”,不只是在存量里抢份额。
3. 受众优先:先建立受众和信任,再把产品、故事和使命接上。
4. 品牌持续相关性:过去有效的打法会失效,团队需要主动停掉旧动作,寻找新的内容、渠道和叙事。
5. AI 边界:AI 能放大执行,但品味、判断、野心、取舍和对用户的长期理解仍要由人负责。

**可沉淀 SOP**
- 00 节点增加“受众优先、产品第二”的战略检查:目标人群、问题、信任资产、内容入口、产品承接必须同表评估。
- 05 节点把“粉丝/受众”与“流量/顾客”区分开,避免只看短期 CPA。
- 92 节点将“停止旧动作”的复盘问题纳入月度经营会:哪些曾经有效的渠道、素材、页面和优惠已经拖累增长。
