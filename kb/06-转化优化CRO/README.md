# 📈 转化优化 CRO

> **节点定位**:把流量“榨干”:用实验把每一份访问变成更多订单。

## 1. 本节点要解决什么
通过 PDP 优化、A/B 测试与购物车/结账优化提升转化率,建立以数据驱动的持续优化机制。

## 2. 核心子主题
- PDP / 落地页转化要素优化
- A/B 测试与实验体系(如 Buy it now 按钮)
- 购物车 / 结账 / 弹窗优化
- 大流量页面转化数据分析
- 加购/赠品/限时等促销机制
- 转化提升路线图与规划

## 3. 真实业务细节(来源:数字化中心周报)
<!-- enrich:周报 -->
**PDP/详情页实验**:Buy it now / Buy now 按钮 AB test(momcozy 详情页已上线)、PDP 转化提升分析专题(BS03)、商详页信息层级优化、批量关闭左下角弹窗视频〔周报02/03·梁庆玲、张乐平、易树炎〕。

**购物车/结账**:购物车与迷你购物车赠品/满赠逻辑统一(基于 cart transform function)、购物车快照与价格解析(便于数据追踪审计)、结账页自动应用折扣、checkout validation function〔周报03·胡凡/体验研发组、林嘉恒、陈进〕。

**流量诊断**:吸奶器品类流量转化分析、IP 识别转化率优化方案调研〔周报03·林龙/数据组、陈进/转化增长组·2026-06〕。

> **对自动化的启示**:转化增长组 + 体验研发组已把 CRO 工程化(组件化按钮/弹窗/赠品 + 数据分析),AB 实验 + Shopify Functions 是自动化优化的抓手。

## 4. 发力点(来源:Twitter 书签)
<!-- enrich:twitter -->
> 书签中无直接 CRO 实验打法;转化实验可纳入运营 Agent 的实验闭环(见 [[10-自动化编排]])。

## 5. Shopify 官方能力 & AI 工具
<!-- enrich:shopify官方 -->
**Functions / Checkout Extensibility 做转化**:满赠、Buy now、结账校验、自动应用折扣等(对应周报)均可用 Shopify Functions 实现,核心结账保持原生〔Shopify 官方〕。
**Sidekick Pulse**:后台主动分析,发现转化/增长机会并给建议〔Shopify 官方〕。
> 发力点:把 CRO 闭环做成「Pulse 发现机会 → Functions 落地 → 数据复盘」。

## 6. SOP 与自动化要点
流程:Pulse 发现机会 → 设计实验 → Functions 落地 → AB 测试 → 复盘。**可自动化**:Pulse 洞察、Functions 配置、实验数据;**人审**:实验上线。
