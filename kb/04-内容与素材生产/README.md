# 🎨 内容与素材生产

> **节点定位**:规模化“造素材”:主图、A+、视频、配音、配乐的AI流水线。

## 1. 本节点要解决什么
用 AI 工具链规模化生产主图/A+、广告视频、配音与配乐,支撑营销与PDP的素材供给。

## 2. 核心子主题
- 主图与 A+ 内容(Midjourney / imini / Canva / Photoshop)
- 商品/场景图与AI图像生成
- 广告短视频剪辑(CapCut / 剪映 / Runway / Kling)
- AI 配音(ElevenLabs)与多语口播
- 背景音乐/氛围音(Suno)
- UGC 与红人素材的采集与授权

## 3. 真实业务细节(来源:数字化中心周报)
<!-- enrich:周报 -->
**素材中心/素材库**:自建广告素材库(随 Meta 广告系统),支持目录文件夹拖拽/重命名、多图与视频预览、素材增删改〔周报01·刘兵兵/平台研发组、周文军/营销产品组·2026-03〕。

**AI 生图/生视频流水线**:图片生成 API 集成(nano banana 模型)、图片扩图(seedream5.0 模型)、视频生成工作流搭建与批量调用〔周报01·李礼/应用算法组·2026-03〕;引入「AI 广告素材制作工具」与「AI 视频分析与混剪」〔周报01·廖志龙/营销产品组·2026-03〕。

**营销 AI 组**:视频中心 1.0/2.0(视频翻译、视频生成),TikTok 广告视频分析(G 3.5 flash 批量接口)〔周报03·魏诺亚/营销AI组·2026-06〕。

> **对自动化的启示**:素材生产已「工作流化 + 模型化」,与你 AI 开店图里的主图/A+/视频/配音工具链天然对接,可直接纳入自动化编排(见 [[10-自动化编排]])。

## 4. 发力点(来源:Twitter 书签)
<!-- enrich:twitter -->
**AI 短视频「情绪公式」流水线**:爆款 AI 短片教程拆出「爆款情绪公式 / 6 宫格故事板 / Seedance 分段视频提示词 / 60 秒成片结构」,核心观点「AI 视频真正难的不是把画面做漂亮,而是把情绪做准确」〔Twitter @joshesye〕。
> **发力点**:把广告视频做成「脚本 → 分镜 → 图生视频(Seedance/Kling)→ 配音 → 成片」模板化流水线,以「情绪公式」为质量标尺。
**素材采集**:universalDownloader 一键下载 Threads/TikTok/IG/YouTube 等 18+ 平台视频〔Twitter @iluciddreaming〕,可用于竞品/UGC 素材采集(版权见 [[91-合规与风控]])。

## 5. Shopify 官方能力 & AI 工具
<!-- enrich:shopify官方 -->
**Sidekick 出图**:按提示词一次生成最多 5 张图(如「夏季促销 banner」),可改图并存入媒体库〔Shopify 官方 · Winter '26〕。
**Shopify Magic 文案**:标题/邮件主题/描述一键生成〔Shopify 官方〕。
> 发力点:店内素材(banner/主图/文案)优先走 Sidekick/Magic;复杂广告视频走外部工具链(见 [[90-AI能力地图]])。

## 6. SOP 与自动化要点
流程:需求 → 脚本/情绪公式 → 图生图/图生视频 → 配音/配乐 → 审片。**可自动化**:Sidekick/Magic 出图、视频工作流;**人审**:品牌调性/版权。
