## 2. DTC 增长总模型

源资料的共同主线是:独立站不是一个孤立商城,而是品牌出海的用户资产与利润控制中枢。可迁入 ABI 的总模型为:

`市场洞察 -> 品牌定位 -> 内容矩阵 -> 红人/联盟/社媒协同 -> 独立站承接 -> 广告放量 -> 大促协同 -> 数据复盘 -> 组织 SOP`

关键判断:
- 市场洞察要先识别赛道容量、竞争烈度和需求成熟度,再决定是做宽品类、垂类分支还是单品突破。〔来源:platform-operations-wiki · E098〕
- 品牌定位要回答“卖给谁、解决什么问题、为什么是你”,避免只写内部自嗨的品牌故事。〔来源:platform-operations-wiki · E098〕
- 内容、达人、投放和独立站页面必须共享同一套人群、卖点、场景和风险词,否则前端流量与站内承接会脱节。〔来源:platform-operations-wiki · E098/E040〕
- Amazon、TikTok Shop 与 Shopify 独立站可以协同,但每个平台的规则与归因口径必须分开,不能混用。〔来源:platform-operations-wiki · E040/E105〕

## 3. 市场洞察与品牌表达

ABI 的 `00-战略与定位` 节点应把品牌战略输出成可供后续 Agent 调用的结构化字段:

| 字段 | 用途 | 进入下游节点 |
|---|---|---|
| target_market | 明确国家/语言/渠道优先级 | 01/05/07 |
| core_audience | 定义人群标签、购买动机、使用场景、决策障碍 | 01/03/04/06 |
| pain_itch_gain | 区分痛点、痒点和实际收益 | 03/04/06 |
| brand_positioning | 一句话定位、差异化价值、品牌故事 | 02/04/05 |
| channel_role | Meta/TikTok/Google/SEO/Reddit/YouTube/联盟的阶段角色 | 05/07 |
| manual_review_gate | 预算、广告上线、功效宣称、素材授权和大促折扣的人审点 | 10/91 |

落地时不要让 Agent 只输出“适合做某类产品”。更可靠的输出应包括:
- 为什么现在做:趋势、需求缺口、竞争空档。
- 用户在哪里表达需求:搜索、评论、社媒、Reddit、YouTube、竞品评论。
- 当前供给缺口是什么:功能、情绪、价格、视觉、服务或内容表达。
- 验证路径是什么:小样、内容测试、落地页测试、广告小预算、KOL 反馈、SEO/GSC 数据。

## 4. 内容到转化的协同链路

DTC 全域内容营销不是多发内容,而是让内容在不同节点承担不同任务:

| 内容层 | 目标 | 典型资产 | 承接方式 |
|---|---|---|---|
| 品牌内容 | 建立记忆点和可信度 | 品牌故事、创始人表达、价值观、场景大片 | 首页、About、社媒主页 |
| 种草内容 | 解释需求和使用场景 | TikTok/Instagram/YouTube Shorts、UGC、测评 | PDP、落地页、邮件、广告素材 |
| 转化内容 | 降低购买阻力 | 对比图、FAQ、评论、教程、折扣、限时活动 | PDP、购物车、EDM、再营销 |
| 复盘内容 | 沉淀可复用经验 | 爆款要素、失败素材、达人合作记录 | 07 数据归因、92 组织 SOP |

对 ABI 的含义:
- `04-内容与素材生产` 需要继承 `00` 的定位字段,避免素材与品牌表达脱节。
- `06-CRO` 的首页、PDP 和落地页要根据渠道意图做差异化承接。
- `07-数据与归因` 要把自然内容、付费广告、红人、联盟和邮件一起纳入复盘,不能只看单渠道 ROI。

## 5. 大促与跨渠道节奏

源资料把 Shopify、社媒和 Amazon 的大促协同拆成预热、爆发和返场三阶段。迁入 ABI 后,可抽象为:

- 预热期:统一主题、页面、内容日历、红人排期、广告素材和库存/客服准备。
- 爆发期:监控流量、转化、库存、支付异常、客服压力和广告预算节奏。
- 返场期:复盘渠道效率、沉淀用户资产、清理库存、再营销和会员转化。

关键红线:
- 不因某个平台促销而绕过 Shopify 独立站的价格、物流、退款和客服承诺。
- 不把 Amazon/TikTok 的平台内玩法直接迁移为独立站动作。
- 不把课程案例中的销售数字写入 ABI 目标或验收。

## 6. 入库结论

本资料包补强的是 `00 战略与定位` 的顶层经营框架:从“建一个 Shopify 店”升级为“构建 DTC 增长系统”。后续节点应统一使用同一套目标人群、品牌表达、渠道角色、素材标签和人审闸,否则独立站 AI 自动化会在局部提效,但无法形成经营闭环。
