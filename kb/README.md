# 🏪 ABI 智能化独立站 · Shopify AI 全链路经营知识库

> **ABI 智能化独立站** — 把一家 Shopify 独立站从「人肉运营」升级为「AI 自动化经营」:降本 · 提效 · 可复制 · 可治理。**一图看懂产品 + 知识库总架构** → [`_diagrams/00_ABI总架构_architecture.png`](_diagrams/00_ABI总架构_architecture.png)。

本知识库服务于一个目标:**用全 AI 自动化的方式经营一家 Shopify 独立站**——从一键式建站到自动化运营全流程。

知识围绕**“建站 → 全自动运营”的全流程阶段**组织(顶层分类),工具与团队作为标签横切。每篇文档同时面向**人阅读**与 **AI 检索/Agent 调用**(统一 frontmatter 元数据 + 可切块结构)。
