# 🤝 客户与会员运营

> **节点定位**:把客户“留住”:会员、忠诚度、社区与口碑的长期经营。

## 1. 本节点要解决什么
经营客户全生命周期:会员中心/SCRM、忠诚度与权益、社区与VOC、舆情监控,提升复购与口碑。

## 2. 核心子主题
- 会员中心 / SCRM 与账户体系(含多区域会员)
- 忠诚度 / 积分 / 礼品卡与权益
- 社区运营(如 communitify 社区)
- VOC 与用户声音回收
- 舆情监控(品牌/独立站社区/客服高危识别)
- 客服与体验(含 AI 客服)

<!-- enrich:周报 -->
**会员中心**:momcozy 会员中心,支持会员全球化(用户认证)、多店铺(订单/会员/优惠券/礼品卡适配)、会员成长值/任务/等级/奖励策略、东南亚站点第三方登录〔周报01·杨醒/服务研发组、吴雨宁/营销产品组·2026-03〕。

**客服/社区**:在线客服、独立站 SaaS 社区内容推荐算法调研〔周报01·陈晨、杨醒〕。

## 5. Shopify 官方能力 & AI 工具
<!-- enrich:shopify官方 -->
**Agentic Storefront / Store AI**:店面自带 AI 回答买家问题、推荐商品、处理订单咨询,基于目录/库存/政策数据,商家控制语气与升级策略〔Shopify 官方 · Spring '26〕。
**Storefront MCP**:search_shop_policies_and_faqs、get_order_status 等支撑 AI 客服〔Shopify 官方〕。
**UCP + Shop Pay**:买家可在 AI 对话内完成购买〔Shopify 官方 · 与 Google 共建 UCP〕。
> 发力点:用 Store AI + MCP 搭「AI 客服 + 会员问答」,与周报会员中心/舆情(§3)打通。
