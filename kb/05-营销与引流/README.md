# 📣 营销与引流

> **节点定位**:把人“引进来”:付费、自然、邮件、红人、活动多渠道协同获客。

## 1. 本节点要解决什么
搭建多渠道获客与召回体系:付费广告、SEO/SEM、邮件营销、红人合作、活动页与玩法,形成可衡量的流量引擎。

## 2. 核心子主题
- 付费广告(Google Ads / Meta / TikTok)与SPU映射
- SEO / SEM 与内容营销
- 邮件 / 短信营销与自动化流(Klaviyo 类)
- 红人 / KOL 合作全流程
- 活动页与玩法(促销页/抽奖/签到/Registry)
- 社媒与社区运营、UTM 投放打标

## 3. 真实业务细节(来源:数字化中心周报)
<!-- enrich:周报 -->
**活动玩法**:通用签到、母亲节互动(做任务攒能量)、Mama Registry / baby registry、春促抽奖页面模板化、秒杀/妇女节活动取数〔周报01/02·方汉涛/平台研发组、廖志龙、梁庆玲〕。

**付费广告中台**:自建广告系统(一期/二期),含 Meta 广告授权与报告、AMZ 广告 SPU 映射、TikTok 数据(挂链/紫鸟浏览器 cookie 获取)、广告标签管理、列表自定义视图〔周报01·周文军、蔡尚辰、农昊天〕。

**邮件/弹窗**:多国 Omnisend 邮件模板 + Klaviyo 弹窗注册搭建(西/德/法/波/英/中东),Shopify 独立站缺货等异常场景邮件通知〔周报03·蔡尚辰 / 周报01·孔文静〕。

**红人营销**:红人系统/红人审计、红人资源与合作数据、红人曝光/播放量数据治理与授权〔周报01·林龙、谭鹏、姚湘〕。

**落地页**:母亲节品牌页、瑜伽球大单品页等落地页设计与上新〔周报01·梁庆玲/站点产品组·2026-03〕。

> **对自动化的启示**:已有「活动+广告+邮件+红人+落地页」五件套与中台数据;与 UTM/归因([[07-数据与归因]])打通后即可做闭环投放自动化。

## 4. 发力点(来源:Twitter 书签)
<!-- enrich:twitter -->
**AI 投放「操作系统」与审核 Skill**:书签出现「Claude Ads —— 付费广告运营系统 / 广告审核技能」的玩法〔Twitter @XAMTO_AI〕。
> **发力点**:把广告账户审核、否词、预算分配、素材轮换做成 Agent skill,与周报的自建广告中台(§3)对接,形成「投放-审核-优化」闭环。
**信息→营销**:Horizon 式信息雷达可反向用于「热点借势 + 自动生成营销简报/选题」〔Twitter @iluciddreaming〕。

## 5. Shopify 官方能力 & AI 工具
<!-- enrich:shopify官方 -->
**Campaign Autopilot(AI 投放,核心)**:在 Admin 内设预算、审批上线,AI 自动跨渠道建活动、分配预算并持续优化——覆盖 Meta、Shop、邮件,并陆续接入 ChatGPT Ads、Microsoft Advertising(7 月)、Snapchat〔Shopify 官方 · Spring '26〕。
**Shopify Magic**:邮件主题行等营销文案。
> 发力点:把 Campaign Autopilot 作为「AI 投放主控台」,与周报自建广告中台(§3)互补,人审守门。


> **【2026 深化】** **Sidekick App Extensions 已上线**,Klaviyo、Loop 等已接入——可在 Sidekick 对话内直接触发邮件/营销动作。详见 [代理式商务技术栈](../02-建站与基础设施/Shopify代理式商务技术栈2026.md)。

## 6. SOP 与自动化要点
流程:目标 → Campaign Autopilot 建活动 → 预算分配 → 优化 → 复盘。**可自动化**:Autopilot 投放、邮件流、否词;**人审**:预算闸/素材合规。

## 8. Inbox 独立站实操资料包补充(2026-06-27)
<!-- enrich:inbox -->
新增资料把站外增长拆成“渠道角色 + 阶段 + 风控 + 数据回收”:FB 群组负责低成本自然触达,Slickdeals 负责价格敏感流量和短期排名推动,KOL/KOC/UGC 负责内容资产和信任,ACC/Affiliate 更接近成交付费型渠道,Reddit 负责社区口碑与 Google 长尾,Meta/Google 负责冷启动测试与规模化扩量〔来源:亚马逊站外全域流量增长;独立站从0-1搭建高流量店铺;独立站网红+全域社媒流量增长〕。

站外投放前应一次性准备目标、素材、落地页、UTM、库存、折扣、客服、评论承接和复盘口径。渠道动作不能脱离 `91-合规与风控`,尤其不能把刷票、多账号投票、虚假折扣或硬广灌水包装成自动化〔来源:Slickdeals使用指南;Facebook矩阵长效养号完整攻略〕。
