# 🛡️ 合规与风控

> **节点定位**:贯穿全程的“红线与护栏”:母婴品类与跨境合规不可踩。

## 1. 本节点要解决什么
沉淀母婴品类与跨境经营的合规要求与风控要点:产品安全/认证、广告合规、隐私与数据、支付风控、内容版权。

## 2. 核心子主题
- 母婴产品安全与区域认证
- 广告与宣称合规(医疗/功效类措辞)
- 隐私与数据合规(GDPR/CCPA/Cookie)
- 支付风控与拒付
- 内容/素材版权与红人授权
- 税务与海关合规
- [站外渠道风控 SOP](站外渠道风控SOP.md)

## 3. 真实业务细节(来源:数字化中心周报)
<!-- enrich:周报 -->
**内容/下载合规**:CMS【免责声明】经销商等外部人员下载风险规避(需求→开发→上线全流程)〔周报01/02·廖志龙、周桥生、黄杰海〕。

**授权与认证**:Meta 广告授权账户/账号列表管理、GA 审计、会员全球化用户认证 + 全球认证网关 + mtls 跨洋调用、ERP-认证管理(合规认证线上化)〔周报01·周文军、杨醒、张莹〕。

**数据采集合规风险(需警惕)**:紫鸟浏览器多店铺 cookie 获取、RPA 取 cookie——属灰色采集,需评估各平台 ToS 与封号风险〔周报01/02·农昊天、徐宗袁〕。

**红人版权**:红人视频仅同步已授权内容〔周报01·廖志龙·2026-02〕。

> **对自动化的启示**:合规已涉及内容免责、广告授权、数据隐私/认证、采集风险、红人版权五类红线,必须在自动化流程中内置「护栏」。

## 4. 发力点(来源:Twitter 书签)
<!-- enrich:twitter -->
**书签暴露的三类合规信号**:
- **平台 API 权限边界**:Shopify 把后台读写开放给 Agent,意味着「Agent 误操作/越权」风险上升,需做权限分级与操作审计〔Twitter @AYi_AInotes、@Shopify〕。
- **素材版权 / 平台 ToS**:一键下载多平台视频(universalDownloader)便利但涉及版权与各平台条款〔Twitter @iluciddreaming〕。
- **数据爬取合规**:信息雷达/Reddit 抓取需遵守 robots 与平台规则〔Twitter @iluciddreaming、@MindOS_Lisa〕。
> **发力点**:在自动化流程中内置「权限护栏 + 版权校验 + 采集合规」三道闸。

## 5. Shopify 官方能力 & AI 工具
<!-- enrich:shopify官方 -->
**代理式商务的新治理点**:Campaign Autopilot 采「设预算 → 审批上线 → AI 执行」的护栏模式;Store AI 由商家控制语气与升级;开放 MCP 写权限要求严格的「Agent 操作审计与权限分级」〔Shopify 官方〕。
> 发力点:对所有 Agent 写操作(改价/上下架/退款/投放)设审批闸与日志,沿用 Autopilot 的「人审守门」范式。

## 6. SOP 与自动化要点
流程:权限分级 → 操作审计 → 版权/采集校验 → 审批闸。**可自动化**:日志与告警;**人审**:红线操作(医疗功效措辞、儿童安全、退款、价格)。
